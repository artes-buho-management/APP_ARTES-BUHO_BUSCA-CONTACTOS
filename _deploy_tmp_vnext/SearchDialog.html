🦉 BUSCA CONTACTOS EN INTERNET

El proceso usa scraping por codigo + IA local potente (IA RUBEN COTON) con trazabilidad completa. Veras la consola en vivo con detalle de cada paso. Se crea/actualiza la pestana TRAZA_OBTENCION con la URL y evidencia de cada contacto guardado.

Modo ahorro real: el motor principal es endpoint de IA local; Gemini queda desactivado por defecto.

Modo estricto activo: si incluyes EMAIL, TELEFONO, NOMBRE CONTACTO o EMPRESA, esos datos se validan siempre en web.

Sin columnas obligatorias: solo se exige en modo estricto la columna que tu incluyes en esta busqueda.

Email profesional: se descartan emails genericos tipo info@ o contacto@.

Email mas estricto: tambien se filtran buzones de rol tipo comunicacion@, prensa@, media@ o events@.

Auditoria email estricta: el correo pasa validacion profunda (web + dominio + DNS/MX).

Contraste de dominio: el dominio del email se cruza con web/fuente y contexto de entidad para evitar mezclas.

Segundo plano: aunque cierres este popup, el proceso sigue ejecutando y luego puedes reanudar estado.

Modo ULTRA: auditoria profunda activa; se permite @gmail/@outlook si el correo esta verificado en web real.

Modo rapido: primero valida la URL principal y solo abre mas paginas si faltan pruebas.

Telefono opcional real: si no pides TELEFONO, no se busca ni se valida telefono.

Control de calidad: minimo 3 comprobaciones web por contacto. Si falta un dato de columna se escribe "IA no encuentra".

Anti-duplicados: se comprueba el email en tiempo real durante la busqueda y no se guardan repetidos.

Auto-reintento activo: si una ventana de tiempo se agota, el proceso continua solo sin detenerse.

Memoria activa: el popup recuerda automaticamente tu ultima busqueda y columnas.

IA local potente: usa Agresivo caudal para volumen maximo o Calidad maxima para filtro estricto.

Cada fila tiene dos celdas: 1) cabecera que veras en la hoja, 2) descripcion de esa cabecera (ejemplo: correo, telefono, web, empresa).
